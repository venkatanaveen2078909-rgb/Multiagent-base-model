# `agents/base.py` — Line-by-Line Explanation

**File:** `agents/base.py`  
**Role:** Defines the abstract base class `BaseAgent` that all specialist agents inherit from. It also defines the standard `AgentOutput` data structure returned by every agent.

---

## AgentOutput Dataclass (Lines 30–41)

```python
@dataclass
class AgentOutput:
    """Standardised output returned by every agent."""
    agent_name: str
    version: str
    prompt_version: str
    task: str
    result: str
    metadata: Dict[str, Any] = field(default_factory=dict)
    latency_ms: float = 0.0
    success: bool = True
    error: Optional[str] = None
```

- **`@dataclass`**: Decorator that automatically generates `__init__`, `__repr__`, etc. for the class.
- **`agent_name` & `version`**: Identifies which agent produced this output and its code version.
- **`prompt_version`**: The specific version of the system prompt used (e.g., "v2"). Crucial for A/B testing and evaluation.
- **`task`**: The user input or prompt that the agent processed.
- **`result`**: The actual text generated by the LLM.
- **`metadata`**: A dictionary for agent-specific data (e.g., detected language, search depth).
- **`latency_ms`**: How long the LLM call took.
- **`success` & `error`**: Indicates whether the execution completed successfully or encountered an exception.

## Constructor `__init__` (Lines 57–75)

```python
    def __init__(
        self,
        prompt_version: Optional[str] = None,
        temperature: Optional[float] = None,
        max_tokens: Optional[int] = None,
    ):
        settings = get_settings()

        self.llm = ChatGroq(
            api_key=settings.groq_api_key,
            model=settings.groq_model,
            temperature=temperature or settings.groq_temperature,
            max_tokens=max_tokens or settings.groq_max_tokens,
        )

        self.prompt_record = _registry.get(self.agent_name, prompt_version)
        self.log = structlog.get_logger().bind(agent=self.agent_name)
```

- **`get_settings()`**: Retrieves the application configuration from `.env`.
- **`ChatGroq(...)`**: Initializes the LangChain LLM wrapper for Groq. It uses default settings unless overridden by the constructor arguments.
- **`_registry.get(...)`**: Fetches the correct `PromptRecord` from the `PromptRegistry`. If `prompt_version` is None, it gets the default version.
- **`structlog.get_logger().bind(...)`**: Creates a logger specifically bound to this agent's name, so all logs automatically include `agent="<name>"`.

- **`@abc.abstractmethod`**: Forces any subclass to implement this method. If a subclass doesn't implement `run()`, Python will raise an error when trying to instantiate it.
