## Node: Evaluator (Lines 209–263)

```python
async def evaluator_node(state: GraphState) -> GraphState:
    """Evaluate the specialist agent's output and build the final report."""
    agent_output: AgentOutput = state["agent_output"]

    if not agent_output or not agent_output.success:
        # Handle early failure gracefuly
        return { **state, "final_report": f"❌ Agent execution failed..." }

    evaluator = EvaluatorAgent()
    evaluation = await evaluator.evaluate(agent_output)
```

- Invokes the `EvaluatorAgent` on the output generated by the specialist node.
- Constructs a massive `final_report` markdown string that stitches together the original agent's answer alongside a cleanly formatted table showing the evaluation metrics and certification pass/fail status.
- Attaches `evaluation` and `final_report` to the graph state.

## Conditional Routing Edge (Lines 269–277)

```python
def route_to_agent(state: GraphState) -> str:
    """Conditional edge: supervisor → research | career | code_review."""
    agent = state.get("selected_agent", "research_agent")
    valid = {"research_agent", "career_agent", "code_review_agent"}
    if agent not in valid:
        return "research_agent"
    return agent
```

- This isn't a node; it's a LangGraph routing function. It looks at the state after the supervisor finishes and tells LangGraph the string name of the next node to jump to.

## Graph Construction (Lines 283–321)

```python
def build_graph() -> StateGraph:
    builder = StateGraph(GraphState)

    # Add nodes
    builder.add_node("supervisor", supervisor_node)
    builder.add_node("research_agent", research_node)
    # ... other agents ...
    builder.add_node("evaluator", evaluator_node)

    # Wire it together
    builder.add_edge(START, "supervisor")

    builder.add_conditional_edges(
        "supervisor",
        route_to_agent,
        {
            "research_agent": "research_agent",
            "career_agent": "career_agent",
            "code_review_agent": "code_review_agent",
        },
    )

    # All specialist agents converge on the evaluator
    builder.add_edge("research_agent", "evaluator")
    builder.add_edge("career_agent", "evaluator")
    builder.add_edge("code_review_agent", "evaluator")

    # Final step
    builder.add_edge("evaluator", END)

    return builder.compile()
```

- Uses `StateGraph` from LangGraph to physically map out the data flow.
- Defines the nodes using the functions created above.
- Defines the edges:
  - `START` points to the `supervisor`.
  - `supervisor` uses conditional routing to branch.
  - All branches converge back into the `evaluator`.
  - `evaluator` points to `END`.
- `builder.compile()` finalises the graph into an executable object.
